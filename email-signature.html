How to use this signature

  1. Select all the content in the Preview box below and copy it (Ctrl+A then Ctrl+C)
  2. Gmail: Settings → See all settings → General → Signature → Create new → paste
  3. Apple Mail: Preferences → Signatures → click + → paste in the right panel
  4. Outlook: Settings → View all settings → Compose and reply → Email signature → paste
